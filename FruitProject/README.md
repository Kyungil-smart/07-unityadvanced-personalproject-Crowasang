# Watermelon Merge 2D
Unity 최신 시스템과 최적화 패턴을 적용한 물리 기반 머지 퍼즐 게임

## 프로젝트 개요
- 엔진: Unity (2D)
- 입력: Input System (키보드)
- 구조: MainScene(메인 메뉴) → GameScene(인게임)

## 개발 환경
- Unity 6000.3.91f1 LTS
- Universal 2D

## 게임 소개
동일한 과일을 합쳐 더 큰 과일로 진화시키는 2D 물리 퍼즐 게임입니다.

## 조작 방법 (Controls)

| 입력 장치 | 조작 방식 | 기능 |
| :--- | :--- | :--- |
| **마우스 (Mouse)** | 좌우 이동 | 과일 투하 위치 조절 |
| | 왼쪽 클릭 | 과일 투하 (Drop) |
